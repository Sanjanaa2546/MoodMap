# MoodMap 🗺️
> Tell me your mood. I'll tell you where to go.

A full-stack AI-powered web app that recommends places based on your mood, city, budget, and time.

🔗 **Live Demo:** https://moodmap-n5iq.onrender.com

---

## Features
- 🧠 Mood-based place recommendations (Relaxed, Foodie, Adventure, Nature, Creative)
- 🤖 AI-generated visit suggestions using Groq LLaMA3
- 🗺️ Live interactive map with exact location pins (Leaflet.js)
- 🔍 Real-time search by food type, place name, or tags
- 💾 Save favourite places locally
- 🏙️ 110 curated places across 18 Indian cities

## Tech Stack
| Layer | Technology |
